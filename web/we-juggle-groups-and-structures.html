48. We juggle groups and structures.

Almost all modern adults are members of multiple groups, and multiple sorts of groups (family, friends, neighbors, church, work, civil society, and so on).

And almost all modern organizations utilize multiple organizing structures, of the basic types described above.

And each of us, as individuals, need to juggle these different groups and organizing structures in order to live our modern lives.
